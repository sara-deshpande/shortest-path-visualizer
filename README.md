# Shortest Path Visualizer

An interactive graph visualization tool built with vanilla HTML, CSS, and JavaScript.

## Features
- Create custom graphs by clicking to place nodes and connecting them with edges
- Supports directed, undirected, and weighted graphs
- Visualizes three shortest path algorithms:
  - Dijkstra's Algorithm
  - Breadth-First Search (BFS)
  - Bellman-Ford Algorithm
- Step-by-step playback of algorithm execution so you can follow along
- Graph is drawn on an HTML canvas for smooth interactive rendering

## How to Run
No installation needed! Simply open `shortest_path.html` in any web browser.
